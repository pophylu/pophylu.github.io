利用Github搭建自己的个人博客

1、安装git

官方地址：https://git-scm.com/

git --version

2、安装node.js

官方地址：https://nodejs.org/en/download/

node -v

3、安装Hexo

npm install hexo -g

hexo -v

4、安装hexo依赖

npm install --save hexo-deployer-git

5、github配置ssh key

6、开始之前先介绍一下heox的基本命令

7、新建一个保存博客的目录，例如E:\worksource\blogger

进入到博客目录并初始化博客目录

hexo init

8、生成静态网页

hexo g

9、预览

hexo s

浏览器输入http://localhost:4000/进行本地预览

10、部署到github

新建一个github仓库

pophylu.github.io

编辑博客目录下的_config.yml

发布到github

hexo d

浏览器访问Hexo (pophylu.github.io)

11、编写博客

新建博客

hexo new '博客名称'

生成html静态文件

hexo g

上传到github

hexo d
